# Cloud Foundry CLI

***Cloud Foundry CLI*** é o cliente oficial da linha de comando para [Cloud Foundry] (https://cloudfoundry.org).
Consulte a documentação [docs CLI] (https://docs.cloudfoundry.org/cf-cli).

Para ajuda, executar
`cf help`

# Bluemix
